TCO 2020 Round 2B 1 250 FixedNumberOfDigits

Problem Statement

You are writing down numbers in increasing order, starting with start and increasing by step. You stop after writing down numberOfDigits digits total. You stop immediately, even if it's in the middle of a number.

Calculate and return the last number you wrote.

Definition

  • Class FixedNumberOfDigits
  • Method sum
  • Parameters int , int , long long
  • Returns long long
  • Method signature long long sum(int start, int step, long long numberOfDigits)
(be sure your method is public)

Limits

  • Time limit (s) 2.000
  • Memory limit (MB) 256

Constraints

  • start will be between 0 and 10^9, inclusive.
  • step will be between 1 and 10, inclusive.
  • numberOfDigits will be between 1 and 10^16, inclusive.

Test cases

  1. input
    • start 47
    • step 10
    • numberOfDigits 7
    output
    Returns 7
    note
    You will write down "47, 57, 67, 7" and stop. The last number you wrote down was 7.
  2. input
    • start 98
    • step 1
    • numberOfDigits 10
    output
    Returns 101
    note
    You will write down "98, 99, 100, 101". The last number written down is 101.
  3. input
    • start 0
    • step 1
    • numberOfDigits 7
    output
    Returns 6
  4. input
    • start 123456789
    • step 10
    • numberOfDigits 5
    output
    Returns 12345
  5. input
    • start 123456789
    • step 10
    • numberOfDigits 17
    output
    Returns 12345679
